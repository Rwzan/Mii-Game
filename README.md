# 🎮 صانع شخصيات Mii (Mii Character Maker)

تطبيق ويب تفاعلي وممتع مستوحى من نظام شخصيات **Mii** الشهير من Nintendo. يتيح للمستخدمين تصميم شخصياتهم الخاصة وتخصيصها بالكامل عبر دمج واختيار عناصر مختلفة (الجسم والملابس، تسريحات الشعر، وملامح الوجه) مع إمكانية التحكم الكامل والدقيق في الأبعاد والمحاذاة لتناسب جميع الأذواق بشكل مثالي ومتناسق! ✨

---

## 🚀 المميزات (Features)

* **خيارات تخصيص متعددة:**
    * 👕 **27 خياراً مختلفاً للأجسام والملابس** لتنوع مظهر الشخصية.
    * 💇‍♂️ **28 تصفيفة شعر مميزة** تدعم ألواناً وقصات متنوعة.
    * 👀 **30 ملامح وجه مختلفة** (عيون، فم، وتعبيرات) لتجسيد مختلف المشاعر.
* **تحكم يدوي متقدم (Advanced Tuning):**
    * **لوحة تحكم كاملة بالشعر والرأس:** إمكانية تحريك الرأس بالكامل في الاتجاهات الأربعة (أعلى، أسفل، يمين، يسار) بالإضافة إلى تكبير وتصغير حجم الرأس لتناسب الرقبة والملابس بدقة.
    * **لوحة تحكم بالملامح:** تعديل دقيق لموقع وحجم العيون والفم داخل الوجه (أعلى، أسفل، يمين، يسار، تكبير، تصغير).
* **استجابة عالية وسرعة فائقة:** تم تحسين حجم خطوة التحريك (Step Size) لتجربة سلسة وسريعة جداً عند الاستخدام من خلال شاشات الجوال أو الحواسيب.
* **تصميم أنيق ومنظم:** واجهة مستخدم مريحة للعين، مبهجة، ومنظمة باستخدام تقنيات CSS الحديثة ونظام الطبقات المتداخلة المستقر (`z-index`).

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لبناء هيكل اللعبة والطبقات الأساسية للشخصية.
* **CSS3:** لتصميم الواجهة، الأزرار التفاعلية، وترتيب طبقات الصور الفوقية بشكل متناسق.
* **JavaScript (Vanilla JS):** لبرمجة منطق اللعبة، التنقل بين الخيارات، ومعالجة عمليات التحريك، التكبير، والتصغير الفوري (Real-time Transformation).

## 📁 هيكل المشروع (Project Structure)

```bash
├── index.html          # الهيكل الرئيسي للموقع ولوحات التحكم
├── style.css           # ملف التنسيق، الألوان، وترتيب الطبقات (CSS)
├── script.js          # ملف البرمجة الذكية، دوال التغيير والتحريك (JS)
└── assets/            # مجلد الوسائط (الصور المقصوصة)
    ├── body (1).png...# صور الأجسام والملابس (1 إلى 27)
    ├── hair01.png...  # صور تسريحات الشعر (01 إلى 28)
    └── face01.png...  # صور ملامح الوجه والعيون (01 إلى 34)

```
## 💻 طريقة التشغيل محلياً (How to Run)
 1. قم بتحميل المشروع أو عمل Clone للمستودع:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/Mii-Character-Maker.git](https://github.com/YOUR_USERNAME/Mii-Character-Maker.git)
   
   ```
 2. تأكد من وضع جميع الصور داخل مجلد باسم assets بنفس الأسماء والترتيب.
 3. افتح ملف index.html مباشرة في أي متصفح إنترنت (Chrome, Safari, Edge) واستمتع باللعبة!
## 💡 كيف تعمل اللعبة برمجياً؟ (How it works)
تعتمد اللعبة على مفهوم **الطبقات الفوقية المتداخلة (Layered Images)** عبر خاصية الـ CSS position: absolute;. تم ربط طبقة الوجه تلقائياً بحاوية الرأس والشعر (#hair-container)، بحيث عندما يقوم المستخدم بتحريك الرأس أو تكبيره، تتبع الملامح الرأس أوتوماتيكياً للحفاظ على ثبات الوزن، مع إتاحة تحكم منفصل للملامح الداخلية لإعطاء حرية مطلقة في التصميم.
